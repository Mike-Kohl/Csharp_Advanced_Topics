Reflection
==========

Overview
========
- Reflection lets us programmatically investigate assemblies
- Can learn about their structure and internal mechanics
- Can be used to operate on otherwise inaccessible types
- Enables other language features (e.g. Attributes)

System.Type
===========
- abstract class


Activator Class
===========
Contains methods to create types of objects locally or remotely, or obtain
references to existing remote objects.  This class cannot be inherited.
